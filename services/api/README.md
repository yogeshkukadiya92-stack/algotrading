## Current Status

- Paper-first backend scaffolding exists here.
- Live-mode references are safety guards and disabled stubs only.

## Phase Rule

Do not extend this directory with live broker execution in the current phase. Structural cleanup and documentation updates are fine; live trading implementation is not.
